### How it works:

1: Create a sign with the text [Checkpoint] and [Earn Reward]

2: If the player falls from the parkour, they fall into the void.

3: Tap the sign!

4: Jump into the void...

5: The plugin will teleport you to your last checkpoint! BOOM.
